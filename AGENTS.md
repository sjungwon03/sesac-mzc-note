# Project Instructions

- When adding, renaming, moving, or deleting a learning note Markdown file, update the `README.md` learning note list in the same change.
- Keep learning note links grouped by topic directory, such as `java` or `mysql`.
- Before writing or editing documentation, read related existing documents first and match their structure, tone, and terminology.
- Do not include sensitive information such as passwords, meeting IDs, phone numbers, or private email addresses in `README.md`.
