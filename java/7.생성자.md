# 7. 생성자

## 생성자란?

생성자는 객체가 생성될 때 자동으로 호출되어 객체의 초기 상태를 설정하는 특별한 메서드이다.

```java
class Student {
    String name;
    int age;

    Student(String name, int age) {
        this.name = name;
        this.age = age;
    }
}
```

특징:

- 클래스 이름과 생성자 이름은 같다.
- 반환 타입을 작성하지 않는다.
- `new`로 객체를 만들 때 호출된다.
- 필드 초기화에 주로 사용한다.

---

## 기본 생성자

생성자를 하나도 작성하지 않으면 컴파일러가 기본 생성자를 자동으로 추가한다.

```java
class Car {
}

Car car = new Car();
```

직접 생성자를 하나라도 작성하면 기본 생성자는 자동으로 만들어지지 않는다.

```java
class Car {
    Car(String model) {
    }
}

// new Car(); // 컴파일 오류
```

---

## this

`this`는 현재 객체 자신을 가리킨다.

```java
class Member {
    String name;

    Member(String name) {
        this.name = name;
    }
}
```

필드명과 매개변수명이 같을 때 `this.name`은 필드, `name`은 매개변수를 의미한다.

---

## 생성자 오버로딩

매개변수의 개수나 타입을 다르게 하여 생성자를 여러 개 정의할 수 있다.

```java
class Product {
    String name;
    int price;

    Product() {
        this("이름 없음", 0);
    }

    Product(String name) {
        this(name, 0);
    }

    Product(String name, int price) {
        this.name = name;
        this.price = price;
    }
}
```

`this()`는 같은 클래스의 다른 생성자를 호출한다.

주의사항:

- `this()`는 생성자 내부에서만 사용할 수 있다.
- `this()`는 생성자의 첫 줄에 작성해야 한다.

---

## 핵심 정리

- 생성자는 객체 생성 시 자동 호출된다.
- 생성자는 객체의 필드 초기화에 사용한다.
- 생성자를 직접 작성하면 기본 생성자는 자동 생성되지 않는다.
- `this`는 현재 객체, `this()`는 다른 생성자 호출을 의미한다.
