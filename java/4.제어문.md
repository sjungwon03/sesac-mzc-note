# 4. 제어문

## 조건문

조건문은 조건에 따라 실행할 코드를 선택한다.

### if 문

```java
int score = 80;

if (score >= 90) {
    System.out.println("A");
} else if (score >= 80) {
    System.out.println("B");
} else {
    System.out.println("C");
}
```

조건식의 결과는 반드시 `boolean`이어야 한다.

### switch 문

하나의 값에 따라 여러 분기를 처리할 때 사용한다.

```java
int menu = 2;

switch (menu) {
    case 1:
        System.out.println("등록");
        break;
    case 2:
        System.out.println("조회");
        break;
    default:
        System.out.println("종료");
}
```

`break`가 없으면 다음 `case`까지 이어서 실행될 수 있다.

## 반복문

반복문은 같은 작업을 여러 번 실행할 때 사용한다.

### for 문

반복 횟수가 정해져 있을 때 주로 사용한다.

```java
for (int i = 1; i <= 5; i++) {
    System.out.println(i);
}
```

구조:

```java
for (초기식; 조건식; 증감식) {
    실행문;
}
```

### while 문

조건이 참인 동안 반복한다.

```java
int count = 1;

while (count <= 5) {
    System.out.println(count);
    count++;
}
```

### do-while 문

본문을 먼저 한 번 실행한 뒤 조건을 검사한다.

```java
int count = 1;

do {
    System.out.println(count);
    count++;
} while (count <= 5);
```

## 반복 제어

### break

반복문 또는 `switch` 문을 즉시 종료한다.

```java
for (int i = 1; i <= 10; i++) {
    if (i == 5) {
        break;
    }
    System.out.println(i);
}
```

### continue

현재 반복만 건너뛰고 다음 반복으로 이동한다.

```java
for (int i = 1; i <= 5; i++) {
    if (i == 3) {
        continue;
    }
    System.out.println(i);
}
```

---

## 주의사항

- 조건식이 항상 `true`이면 무한 반복이 발생할 수 있다.
- 반복문 안에서는 종료 조건이 변경되는지 확인해야 한다.
- 중첩 반복문은 들여쓰기를 정리해 가독성을 유지한다.

---

## 핵심 정리

- `if`는 범위 조건, `switch`는 값 분기에 적합하다.
- `for`는 횟수 중심 반복, `while`은 조건 중심 반복에 적합하다.
- `break`는 종료, `continue`는 다음 반복으로 이동이다.
