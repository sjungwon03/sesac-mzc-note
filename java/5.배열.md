# 5. 배열

## 배열이란?

배열은 같은 타입의 여러 값을 하나의 이름으로 묶어 관리하는 자료구조이다.

```java
int[] scores = {90, 80, 70};
```

| 특징 | 설명 |
| --- | --- |
| 같은 타입 | 하나의 배열에는 같은 타입만 저장 가능 |
| 인덱스 사용 | 각 값은 0부터 시작하는 번호로 접근 |
| 고정 길이 | 생성 후 길이를 변경할 수 없음 |
| 참조 타입 | 배열 변수는 배열 객체를 참조 |

---

## 배열 선언과 생성

```java
int[] numbers = new int[3];
numbers[0] = 10;
numbers[1] = 20;
numbers[2] = 30;
```

초기값을 바로 넣을 수도 있다.

```java
String[] names = {"Kim", "Lee", "Park"};
```

---

## 배열 순회

배열의 길이는 `length`로 확인한다.

```java
int[] scores = {90, 80, 70};

for (int i = 0; i < scores.length; i++) {
    System.out.println(scores[i]);
}
```

향상된 for 문은 전체 값을 순서대로 읽을 때 편리하다.

```java
for (int score : scores) {
    System.out.println(score);
}
```

---

## 2차원 배열

2차원 배열은 배열 안에 배열이 있는 구조이다.

```java
int[][] matrix = {
    {1, 2, 3},
    {4, 5, 6}
};

System.out.println(matrix[0][1]); // 2
```

순회 예시:

```java
for (int i = 0; i < matrix.length; i++) {
    for (int j = 0; j < matrix[i].length; j++) {
        System.out.println(matrix[i][j]);
    }
}
```

---

## 배열 복사

배열은 길이를 변경할 수 없기 때문에 더 큰 배열이 필요하면 새 배열을 만들고 값을 복사한다.

```java
int[] oldArray = {1, 2, 3};
int[] newArray = new int[5];

System.arraycopy(oldArray, 0, newArray, 0, oldArray.length);
```

---

## 주의사항

- 배열 인덱스는 `0`부터 시작한다.
- 마지막 인덱스는 `length - 1`이다.
- 범위를 벗어난 인덱스에 접근하면 `ArrayIndexOutOfBoundsException`이 발생한다.
- 배열 변수끼리 `=`로 대입하면 값 복사가 아니라 참조 복사가 된다.

---

## 핵심 정리

- 배열은 같은 타입의 여러 값을 저장한다.
- 배열 길이는 생성 후 변경할 수 없다.
- 배열 순회에는 `for`, 향상된 `for` 문을 자주 사용한다.
- 2차원 배열은 행과 열 구조의 데이터를 다룰 때 사용한다.
