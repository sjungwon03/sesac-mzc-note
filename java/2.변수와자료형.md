# 2. 변수와 자료형

## 변수

변수는 값을 저장하기 위해 이름을 붙인 메모리 공간이다.

```java
int age = 20;
String name = "Java";
```

| 용어 | 설명 |
| --- | --- |
| 선언 | 사용할 변수의 타입과 이름을 정함 |
| 초기화 | 변수에 처음 값을 저장함 |
| 대입 | 변수에 값을 저장하거나 변경함 |

---

## 기본 타입

기본 타입은 실제 값을 직접 저장한다.

| 분류 | 타입 | 설명 |
| --- | --- | --- |
| 논리 | `boolean` | `true`, `false` |
| 정수 | `byte`, `short`, `int`, `long` | 정수 저장 |
| 실수 | `float`, `double` | 소수점이 있는 수 저장 |
| 문자 | `char` | 문자 하나 저장 |

### 자주 쓰는 타입

- 정수는 보통 `int`를 사용한다.
- 큰 정수는 `long`을 사용하고 값 뒤에 `L`을 붙인다.
- 실수는 보통 `double`을 사용한다.
- 문자 하나는 `char`, 문자열은 `String`을 사용한다.

```java
int count = 10;
long total = 10000000000L;
double rate = 3.14;
char grade = 'A';
boolean active = true;
```

---

## 참조 타입

참조 타입은 객체가 저장된 위치를 참조한다.

```java
String message = "Hello";
int[] numbers = {1, 2, 3};
```

| 구분 | 기본 타입 | 참조 타입 |
| --- | --- | --- |
| 저장 값 | 실제 값 | 객체의 주소값 |
| `null` 저장 | 불가능 | 가능 |
| 예시 | `int`, `double` | `String`, 배열, 클래스 |

---

## 형 변환

### 자동 형 변환

작은 범위 타입의 값을 큰 범위 타입에 저장할 때 자동으로 변환된다.

```java
int number = 10;
double result = number;
```

### 강제 형 변환

큰 범위 타입의 값을 작은 범위 타입에 저장할 때는 명시적으로 변환해야 한다.

```java
double value = 3.14;
int result = (int) value; // 3
```

---

## 문자열

`String`은 문자열을 다루는 참조 타입이다.

```java
String first = "Hello";
String second = "Java";
String result = first + " " + second;
```

문자열 비교는 `==`가 아니라 `equals()`를 사용한다.

```java
String a = "Java";
String b = new String("Java");

System.out.println(a.equals(b)); // true
```

---

## 주의사항

- 변수는 사용하기 전에 반드시 초기화해야 한다.
- `float` 값에는 `F`, `long` 값에는 `L`을 붙인다.
- 정수 연산 결과는 기본적으로 `int` 이상으로 처리된다.
- 기본 타입 비교는 `==`, 문자열 내용 비교는 `equals()`를 사용한다.

---

## 핵심 정리

- 변수는 값을 저장하는 이름 있는 공간이다.
- 기본 타입은 실제 값을 저장하고, 참조 타입은 객체의 위치를 저장한다.
- 작은 타입에서 큰 타입으로는 자동 변환이 가능하다.
- 큰 타입에서 작은 타입으로는 강제 형 변환이 필요하다.
- 문자열 내용 비교는 `equals()`를 사용한다.
