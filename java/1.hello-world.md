# 1. Hello World

## 핵심 개념

### Java 실행 흐름

```text
소스 코드 작성(.java)
        ↓
컴파일(javac)
        ↓
바이트코드 생성(.class)
        ↓
JVM에서 실행(java)
```

Java는 소스 파일을 바로 실행하지 않고, 먼저 컴파일하여 JVM이 실행할 수 있는 바이트코드로 변환한다.

### Hello World 코드

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}
```

| 구성 요소 | 의미 |
| --- | --- |
| `class HelloWorld` | Java 프로그램을 구성하는 클래스 |
| `main` | 프로그램 실행 시작점 |
| `String[] args` | 실행 시 전달받는 문자열 인자 배열 |
| `System.out.println()` | 콘솔에 값을 출력하고 줄바꿈 |

---

## 컴파일과 실행

### 컴파일

```bash
javac HelloWorld.java
```

컴파일에 성공하면 `HelloWorld.class` 파일이 생성된다.

### 실행

```bash
java HelloWorld
```

실행할 때는 `.class` 확장자를 붙이지 않고 클래스 이름만 사용한다.

---

## 주의사항

- `public class` 이름과 파일명은 반드시 같아야 한다.
- Java는 대소문자를 구분한다. `System`과 `system`은 다르다.
- 문장은 대부분 세미콜론(`;`)으로 끝난다.
- 실행 대상은 `.java` 파일이 아니라 컴파일된 `.class` 파일이다.

---

## 핵심 정리

- Java 프로그램은 클래스 단위로 작성한다.
- `main` 메서드는 Java 애플리케이션의 시작점이다.
- `javac`는 컴파일, `java`는 실행 명령어이다.
- `System.out.println()`은 콘솔 출력에 사용한다.
