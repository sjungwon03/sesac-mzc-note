# 3. 연산자

## 산술 연산자

숫자 값을 계산할 때 사용한다.

| 연산자 | 의미 | 예시 |
| --- | --- | --- |
| `+` | 더하기 | `a + b` |
| `-` | 빼기 | `a - b` |
| `*` | 곱하기 | `a * b` |
| `/` | 나누기 | `a / b` |
| `%` | 나머지 | `a % b` |

```java
int a = 10;
int b = 3;

System.out.println(a / b); // 3
System.out.println(a % b); // 1
```

정수끼리 나누면 결과도 정수로 나온다.

---

## 비교 연산자

두 값을 비교하고 결과로 `boolean` 값을 반환한다.

| 연산자 | 의미 |
| --- | --- |
| `==` | 같다 |
| `!=` | 다르다 |
| `>` | 크다 |
| `>=` | 크거나 같다 |
| `<` | 작다 |
| `<=` | 작거나 같다 |

```java
int age = 20;
System.out.println(age >= 19); // true
```

---

## 논리 연산자

조건식을 연결할 때 사용한다.

| 연산자 | 의미 | 설명 |
| --- | --- | --- |
| `&&` | AND | 둘 다 true이면 true |
| `||` | OR | 하나라도 true이면 true |
| `!` | NOT | true와 false를 반대로 변경 |

```java
int age = 20;
boolean hasTicket = true;

System.out.println(age >= 19 && hasTicket); // true
```

---

## 대입 연산자

변수에 값을 저장할 때 사용한다.

```java
int number = 10;
number += 5; // number = number + 5
number *= 2; // number = number * 2
```

| 연산자 | 의미 |
| --- | --- |
| `=` | 대입 |
| `+=` | 더한 후 대입 |
| `-=` | 뺀 후 대입 |
| `*=` | 곱한 후 대입 |
| `/=` | 나눈 후 대입 |
| `%=` | 나머지 후 대입 |

---

## 증감 연산자

값을 1 증가시키거나 감소시킨다.

```java
int count = 0;
count++;
count--;
```

| 형태 | 설명 |
| --- | --- |
| `++x` | 먼저 증가 후 사용 |
| `x++` | 먼저 사용 후 증가 |
| `--x` | 먼저 감소 후 사용 |
| `x--` | 먼저 사용 후 감소 |

```java
int x = 10;
System.out.println(x++); // 10
System.out.println(x);   // 11
```

---

## 삼항 연산자

간단한 조건에 따라 값을 선택한다.

```java
int score = 80;
String result = score >= 60 ? "pass" : "fail";
```

형식:

```java
조건식 ? true일_때_값 : false일_때_값
```

---

## 핵심 정리

- 산술 연산자는 숫자 계산에 사용한다.
- 비교 연산자는 결과로 `boolean`을 반환한다.
- 논리 연산자는 여러 조건을 연결한다.
- 증감 연산자는 위치에 따라 실행 순서가 달라진다.
- 복잡한 조건은 괄호를 사용해 우선순위를 명확히 한다.
