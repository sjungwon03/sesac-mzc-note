# 15. Standard API

## Java 표준 API란?

Java 표준 API는 JDK에서 기본 제공하는 클래스와 인터페이스의 모음이다.

대표 패키지:

| 패키지 | 설명 |
| --- | --- |
| `java.lang` | 기본 클래스, 자동 import |
| `java.util` | 컬렉션, 날짜, 유틸리티 |
| `java.io` | 입출력 |
| `java.time` | 날짜와 시간 |
| `java.sql` | JDBC |

---

## Object 클래스

모든 Java 클래스의 최상위 부모 클래스이다.

주요 메서드:

| 메서드 | 설명 |
| --- | --- |
| `toString()` | 객체 문자열 표현 반환 |
| `equals()` | 객체 동등성 비교 |
| `hashCode()` | 해시 코드 반환 |

객체의 내용 비교가 필요하면 `equals()`와 `hashCode()`를 함께 재정의하는 것이 일반적이다.

---

## String 클래스

문자열을 다루는 클래스이다.

```java
String text = "Hello Java";

System.out.println(text.length());
System.out.println(text.substring(0, 5));
System.out.println(text.contains("Java"));
```

`String`은 불변 객체이다. 문자열 변경이 많으면 `StringBuilder`를 사용한다.

```java
StringBuilder builder = new StringBuilder();
builder.append("Hello");
builder.append(" Java");
```

---

## Wrapper 클래스

기본 타입을 객체로 다룰 수 있게 하는 클래스이다.

| 기본 타입 | Wrapper 클래스 |
| --- | --- |
| `int` | `Integer` |
| `long` | `Long` |
| `double` | `Double` |
| `boolean` | `Boolean` |
| `char` | `Character` |

```java
int number = Integer.parseInt("10");
Integer value = 20;
```

기본 타입과 Wrapper 클래스 사이의 자동 변환을 오토박싱, 언박싱이라고 한다.

---

## 날짜와 시간

Java 8 이후에는 `java.time` 패키지를 주로 사용한다.

```java
LocalDate date = LocalDate.now();
LocalDateTime dateTime = LocalDateTime.now();
```

| 클래스 | 설명 |
| --- | --- |
| `LocalDate` | 날짜 |
| `LocalTime` | 시간 |
| `LocalDateTime` | 날짜와 시간 |
| `DateTimeFormatter` | 날짜/시간 포맷 |

---

## 핵심 정리

- 표준 API는 JDK에서 제공하는 기본 기능 모음이다.
- 모든 클래스는 `Object`를 상속한다.
- 문자열 내용 비교는 `equals()`를 사용한다.
- 문자열 변경이 많으면 `StringBuilder`가 적합하다.
- 날짜와 시간은 `java.time` 패키지를 사용한다.
