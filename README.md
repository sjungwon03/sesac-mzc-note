# (영등포8기) AWS와 AI를 활용한 MSA 기반 웹 서비스 개발

AWS와 AI를 활용해 MSA 기반 웹 서비스를 개발하는 훈련 과정의 학습 자료와 실습 소스코드를 정리하는 저장소입니다.

## 과정 개요

- Java, 데이터베이스, Spring Boot 기반 백엔드 개발 기초 학습
- AI를 활용한 Front-End 개발 기술 학습
- Linux, Network, Docker, Kubernetes 기반 클라우드 인프라 학습
- Spring Cloud와 AWS 기반 MSA 운영 및 서버리스 개발 학습
- AWS AI 서비스 활용 및 프로젝트 기반 실습 진행
